# STS 434: Applied Statistics for Data Science
## Spring 2027 | Dr. James Okenwa | University of Maine

### Course Description
Application of statistical methods to real-world data analysis problems. Topics include regression, classification, clustering, model evaluation, and data visualization. Students use R for all computational work and complete a semester-long team project.

### Prerequisites
- MAT 228 (Calculus III)
- STS 232 (Principles of Statistical Inference)

### Format
- 2x 75-minute meetings per week (lecture + in-class coding)
- Enrollment cap: 35
- Cross-listed as STS 534 (graduate section)

### Learning Outcomes
Upon successful completion of this course, students will be able to:
1. Formulate statistical models appropriate for structured and unstructured data
2. Implement regression, classification, and clustering methods in R
3. Evaluate model performance using cross-validation and appropriate metrics
4. Interpret and communicate statistical findings to non-technical audiences
5. Apply ethical reasoning to data collection, privacy, and algorithmic fairness
6. Collaborate on a semester-long applied data analysis project

### Required Textbook
- *An Introduction to Statistical Learning* by James, Witten, Hastie, Tibshirani (2nd Edition, free PDF)

### Known Gaps
- No SQL or database querying (students only work with pre-loaded CSV/RDS files)
- No experiment design or A/B testing methodology
- No cloud computing platforms (AWS SageMaker, GCP Vertex AI)
- No reproducible research practices (Git, R Markdown/Quarto used informally)
- No model deployment or MLOps concepts
- Python not taught (R only, despite Python's industry dominance)
- No data pipeline or ETL concepts
