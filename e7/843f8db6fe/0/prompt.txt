The project has multiple different metrics to ballance when making a proposal. I want the user to be able to see these metrics when a proposal is given the the user. The metrics should appear outside of the chat interface so they are visable while scrolling through. They should measure: 1-allignment to the original assignment/class, 2-Allignment to the students goals/personality/carrer, and 3-Professor Acceptance (If there are any reasons the professor would not accept the proposal). If the proposal is updated or refined by the user, the system should reevaluate these metrics and update them to reflect the changed scores.
